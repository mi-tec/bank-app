 - Go - Not yet started

## Run Locally

### Rust

Clone the project

```bash
  git clone git@github.com:mi-tec/bank-app.git && cd bank-app
```

Go to the project directory

```bash
  cd rust-bank-app
```

run

```bash
  cargo run
```
